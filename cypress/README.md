# E2E con Cypress — SauceDemo

Suite mínima y prolija para entrevistas:
- `baseUrl` configurado
- `fixtures` de credenciales
- comando custom `cy.login()`
- specs estables (login y checkout)
- modo headless con videos
- CI en GitHub Actions

## Requisitos
- Node 18+ y npm

npm i
